### Install the project’s dependencies:

### *npm run install*

####

#### *Run the project locally:*

####

#### *yarn dev*

## Open [http://localhost:3000](http://localhost:3000/) with your browser to see the result.

# [Building the Starter Locally (Using the hosted content API)](https://quangnhat2003.github.io/tinaweb-ne/)

Replace the .env.example, with .env

NEXT\_PUBLIC\_TINA\_CLIENT\_ID=\<get this from the project you create at app.tina.io>
TINA\_TOKEN=\<get this from the project you create at app.tina.io>
NEXT\_PUBLIC\_TINA\_BRANCH=\<Specify the branch with Tina configured>

Build the project:

yarn build
